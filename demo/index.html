Getting Started

Use the mgx-pane-navigator directive to created a collection of panes that you can swipe between. For this to work, the directive should be used inside an element that has CSS position: absolute|relative|fixed and overflow:hidden and a defined size.

This is intended for mobile use, but click and drag for swiping left and right will work in the browser. Keep in mind that the normal click+drag behavior is to select text on the screen so you will need to account for that.
